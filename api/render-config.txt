# ===== CHAVES DE API NECESSÁRIAS =====

## Para Ethereum/Polygon - Alchemy (gratuito)
# 1. Vá para https://alchemy.com
# 2. Crie conta gratuita
# 3. Create App → Ethereum Mainnet
# 4. Copie a API Key
# 5. Substitua em ETHEREUM_RPC: SUA_CHAVE_ALCHEMY

## Alternativa - Infura (gratuito)
# 1. Vá para https://infura.io
# 2. Crie projeto
# 3. Copie Project ID
# 4. Use em POLYGON_RPC: SUA_CHAVE_INFURA

## Para Private Key (CUIDADO!)
# 1. Use MetaMask → Account Details → Export Private Key
# 2. Cole SEM o prefixo "0x"
# 3. Certifique-se que tem fundos para deploy
# 4. NUNCA compartilhe esta chave!

## Para JWT Secret
# 1. Gere string aleatória forte
# 2. Use: openssl rand -base64 32
# 3. Ou gerador online confiável
