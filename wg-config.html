📋 Código Para Embed
📚 Como usar o Configurador:
  1. Configure a blockchain: Escolha a rede e cole o endereço do contrato
  2. Busque os dados: Clique em "Buscar Contrato" para carregar informações automaticamente
  3. Personalize o widget: Ajuste título, preço e quantidades conforme necessário
  4. Visualize: Use "Atualizar" para ver o preview em tempo real
  5. Gere o código: Clique em "Gerar" para criar o código embed
  6. Copie e use: Cole o código em sua página web. O embed utiliza a função createxcafeWidget do arquivo wg-widget.js.
