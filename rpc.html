MetaMask – Conectar & Adicionar/Atualizar RPC MetaMask: …

Use este formulário para solicitar à MetaMask a adição/atualização de uma rede (RPC) existente por wallet_addEthereumChain e alternar para ela via wallet_switchEthereumChain.

Será convertido para hexadecimal conforme a especificação.
Para “adicionar outro RPC” inclua o novo endpoint aqui junto com os atuais.
Pronto.

Dicas:
• Se a rede já existir na MetaMask para o chainId informado, ela pode ser atualizada com os novos dados; caso contrário, será adicionada.
• Erro 4902: rede não adicionada; chame wallet_addEthereumChain primeiro.
• Erro 4001: ação cancelada pelo usuário.
