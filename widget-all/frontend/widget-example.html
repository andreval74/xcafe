Demonstração do Widget

Este é um exemplo de como o widget XCAFE aparece em seu site. O widget é totalmente personalizável e se adapta ao design do seu site.

Como Integrar

1
Inclua o Script
Adicione o script do widget em seu HTML
2
Configure o Container
Crie um elemento para o widget
3
Inicialize
Configure as opções e inicialize o widget

Código de Integração

1. Incluir o Script
<script src="js/xcafe-widget.js"></script>
2. HTML do Container
<div id="meu-widget-xcafe"></div>
3. Inicialização JavaScript
// Método 1: JavaScript
const widget = createXCAFEWidget('meu-widget-xcafe', {
    apiKey: 'sua-chave-api-aqui',
    serviceName: 'Meu Serviço Premium',
    serviceDescription: 'Acesso completo ao sistema',
    creditsRequired: 50,
    pricePerCredit: 0.1,
    onSuccess: function(result) {
        console.log('Pagamento realizado:', result);
        // Liberar acesso ao serviço
    },
    onError: function(error) {
        console.error('Erro no pagamento:', error);
    }
});

// Método 2: Data Attributes (Auto-inicialização)
<div id="widget-auto" 
     data-xcafe-widget
     data-api-key="sua-chave-api"
     data-service-name="Serviço Premium"
     data-credits-required="50"
     data-price-per-credit="0.1">
</div>
4. Opções Avançadas
const widget = createXCAFEWidget('container', {
    // Configurações básicas
    apiKey: 'xcafe_1234567890abcdef',
    serviceName: 'API Premium',
    serviceDescription: 'Acesso ilimitado à API',
    creditsRequired: 100,
    pricePerCredit: 0.05,
    
    // Callbacks
    onSuccess: function(result) {
        // result.operationTag - Tag da operação
        // result.creditsUsed - Créditos utilizados
        // result.transactionId - ID da transação
        window.location.href = '/dashboard?access=granted';
    },
    
    onError: function(error) {
        alert('Erro: ' + error.message);
    },
    
    // Personalização
    theme: 'dark', // 'dark' ou 'light'
    showPoweredBy: true, // Mostrar "Powered by XCAFE"
});

Documentação da API

Endpoints Principais
  • GET /api/user/credits Consultar créditos
  • POST /api/widget/process-payment Processar pagamento
  • POST /api/user/purchase-credits Comprar créditos
Parâmetros do Widget
  • apiKey (obrigatório)
    Sua chave de API do XCAFE
  • creditsRequired (obrigatório)
    Quantidade de créditos necessários
  • serviceName (opcional)
    Nome do serviço sendo cobrado
